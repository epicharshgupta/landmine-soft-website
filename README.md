# Landmine Soft Website

A modern and responsive company website built using **React.js** and **Tailwind CSS**.
This project represents a professional SaaS-style website for a software development company.

---

## 🚀 Features

* Modern UI with **Blue + Black + White theme**
* Fully **responsive design**
* **Reusable components** (Navbar, Footer, Cards)
* Multiple pages with React Router
* Smooth hover effects and clean layout

---

## 🧰 Tech Stack

* React.js
* Tailwind CSS
* React Router
* JavaScript
* Vite

## ⚙️ Installation

Clone the repository

git clone https://github.com/YOUR_USERNAME/landmine-soft-website.git

Navigate to project folder

cd landmine-soft-website

Install dependencies

npm install

Run development server

npm run dev
